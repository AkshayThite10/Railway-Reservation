1. Create the database using :
   > sh create_db.sh
   
2. Compile the server and client programs
   > gcc -o serv serv.c
   > gcc -o cli cli.c

3. Run the server and then the multiple client codes
   > ./serv
   > ./cli

4. Create the Admin user on the first run of client
   Enter the pin for creating Admin as "admin"
   
5. Carry out the desired operations
   It may include creations of users,trains,bookings,etc.
